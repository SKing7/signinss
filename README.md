# signinss
如果你正在用ss，各大网站的流量通过签到已经能满足日常google，twitter，youtube的需要了。通过这个扩展可以进行签到拿流量了。

#Usage
在chrome extension中添加signinss.crx

`mac版的chrome。目前已禁止安装非chrome store上的应用了，
所以如要使用，需clone代码，然后在chrome中load unpacked extension选择根目录使用`

##备注：
基于`我是小马甲`开发的一键签到
